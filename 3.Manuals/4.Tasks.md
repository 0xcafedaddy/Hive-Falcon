# Tasks

Tasks 模块下所涉及的内容是 YARN 上的任务详情，包含的内容如下所示：
  * All (所有的任务)
  * Running (正在运行的任务)
  * Finished (已完成的任务)
  * Failed (已失败的任务)
  * Killed (已杀掉的任务)

如下图所示：

![Tasks_Finished](../res/tasks_finished@2x.png)
